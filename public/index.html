SIMILAR

The Ooyala Discovery Similar Videos API returns videos that have related usage and behavior to a reference video. The Similar Videos API returns videos that are identified using the “people who watched this video also watched” approach. This is quite like Amazon recommendations “people who bought these products also bought”.

The Similar Videos API is designed to drive exploration of a content catalog by a user in a personalized fashion, and is not expected to keep the user watching content on the same topic. Videos returned from the API can be restricted to specific topics depending on the context. For example, if the user is on the Sports page of your site, you may want to restrict Similar Videos recommendations to the that context.

TRENDING
Hour Day Week Month

The Ooyala Discovery Trending API can return videos using two approaches. First, the Momentum route returns videos that have seen the most acceleration in views in the specified time period. For example, if one of your videos becomes popular on Reddit or Facebook, it might appear in the Trending-Momentum results, which will help you viewers find videos before they become popular. Once it appears, it will be found by your regular site visitors, reinforcing its popularity. Additionally, your regular viewers are affecting the trending results. When new visitors find your site, good recommendations will make them likely to stay around longer. The Trending API affords great control over what can be displayed. For example, you could define logic similar to the following:

  • If the viewer has not accessed the site for over 30 days, display trending content for the month.
  • If the viewer accessed the site between 8 and 30 days ago, display trending content for the week.
  • If the viewer accessed the site between 1 and 7 days ago, display trending content for the day.
  • If the viewer accessed the site less than 24 hours ago, display trending content for the hour.

POPULAR
Hour Day Week Month

While the Trending-Momentum API can be used to alert your viewers to up-and-coming hits, no one wants to miss out on what everyone is watching. The Trending-Top API will return your most-viewed titles over a specific time frame -- today's hits, best of the week, all-time greatest -- the choice is yours.

RECENT

If you have a large library or add content frequently, you can easily create a playlist of the most recently added content, a real benefit to your loyal viewers who come to your site looking the the latest videos.

PROMOTED

Suppose you have specific content you want to include along with Discovery's recommended titles. This is easily accomplished using Promoted videos, whereby you can specify a specific video, or any video from a collection, to be placed within the Discovery results. In this example, a 'sponsored' video is placed into the 3rd position among the recommendations generated by Ooyala Discovery. A great way to serve up specific content to your audience.
